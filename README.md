# Py2Cpp4OR

Intelligent agent for translating Python optimization models (gurobipy/pyomo) to efficient C++ code.

## Purpose

Bridge the gap between Python-based OR research and C++ production performance. Automates translation while learning and refining mapping rules.

## Features

- Python → C++ translation for mathematical models
- Support for gurobipy and pyomo syntax
- Grammar mapping rules documentation
- Performance benchmarking examples
